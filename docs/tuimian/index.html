关于保研、出国、考研之间的抉择

对于这个问题，曾经思索良久，首先其实从制度上来看，这三者只有出国和保研是可以兼得的，当然可能这样会一定程度上负有道德风险（占了坑位又不保研），但是也仅仅是道德风险。

至于后果的话可能是当年浪费一个名额，下一年也可能会因为这个情况给学院扣减一个保研名额（传说中的惩罚政策），每年放弃保研再出国的人都有，但是惩罚的落实似乎也并不是每年都有。

如果抛弃一些诸如家庭背景、疫情等因素，我认为出国>保研>考研，而如何抉择，必然是在充分考虑各方面因素之后针对自己的情况来做出的最佳选择，当然我所分享的经验可能仅限于金融大类，以及后面愿意从事于金融行业的朋友们参考。

从现在的就业市场认可度来看，本科985+海外硕士的竞争力是非常强的，而且我个人认为的最佳思路应当是直接申请1年制英国硕士，然后拿到offer之后选择gap一年，在gap的这一年里可以利用自己的时间去刷实习、考证等等丰富自己的背景，再到就业市场上拿着一个镀金的学历和丰富的经历，绝对是国内正常硕士毕业朋友们的噩梦。

而且从投入产出比来看，出国也是最有价值的一个选择，你可能会觉得出国要花大几十万，甚至美国一些很好的项目都要花近100万，但是你要非常清晰的明白，现在想挣回这些钱是非常容易的（努力才会容易，没有白嫖的回报）。别说进外资了，这个学历还是可以稳进三中一华的，目前中信和中金两家已经基本对齐（2021年12月），应届生第一年拿60w-70w是非常正常的事情，而且在投行的话这些钱你可以理解为是直接进口袋的，其他的日常生活开销基本都能从公司报销。

写到这里想起一个很有意思的事情，最近工作隔壁组有个领导，我认为是把报销发挥到了极致，这个人所有的生活用品（洗发水、牙膏以及奇奇怪怪的私人用品）全都是在IPO企业报销的，这就比较搞笑，不过也不是不行，毕竟工作性质就是这样。

这样的高薪，你还觉得这样的投入不值得吗？如果你觉得不值得，那说明你的职业规划没有在投行这些范围内。当然如果是其他的工作，那就得另谈了，确实可能挣不回来。

如果对金融行业没兴趣，那其实也应该考虑到的就是学历他其实是一种投资，他给你能带来的是你人生阅历层次的变化，而且是一个从上到下的选择权，我觉得在求学阶段，还是根据自己的实际情况，能去好学校就去好学校，不要因为一些其他的事情荒废自己的未来。

对于保研，我觉得非常之辛苦，想通过保研来去到一个好的学校是一个非常消耗时间和经历的选择，但是也是最锻炼人的一个选择。你一定会发现，你走了这条路付出的比别人多得多，得到的缺并不一定会比别人多。比如当一个海归回来和你竞争，如果背景差不多，大概率你要没，没人关心你过程中付出了多少，现实就是结果导向。

考研的话，因人而异，没有考不上的研，如果没考上，请反思是不是对自己定位不清晰，没有付出努力，不要随大流。
